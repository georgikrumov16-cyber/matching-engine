⚡ Matching Engine — C++ Order Book & Trade Matching System

A high‑performance, fully functional **limit order book + matching engine** written in modern C++20.  
Supports:

- Limit orders  
- Market orders  
- FIFO matching  
- Order cancellation  
- Depth‑of‑book  
- Trade history  
- Engine metrics  
- SAVE / LOAD state  
- REPLAY scripts  
- Logging with timestamps  
- CLI interface  

This project demonstrates **systems programming**, **data structures**, **state management**, and **exchange‑style matching logic** — ideal for C++ and DevOps engineering portfolios.
